# Alert - &lt;st-alert&gt;

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

## Examples

Alerts are available for any length of text, as well as an optional dismiss button. For proper styling, use one of the eight variants.

```html
<st-alert classnames="alert-primary">
  This is a primary alert!
</st-alert>

<st-alert classnames="alert-secondary">
  This is a secondary alert!
</st-alert>

<st-alert classnames="alert-success">
  This is a success alert!
</st-alert>

<st-alert classnames="alert-danger">
  This is a danger alert!
</st-alert>

<st-alert classnames="alert-warning">
  This is a warning alert!
</st-alert>

<st-alert classnames="alert-info">
  This is a info alert!
</st-alert>

<st-alert classnames="alert-light">
  This is a light alert!
</st-alert>

<st-alert classnames="alert-dark">
  This is a dark alert!
</st-alert>
```
